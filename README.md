Little Lemon is a restaurant website developed using the Django Web Framework. This project was created as part of the course on Django Web Framework offered by Coursera.
Features:

    Home: Welcome to Little Lemon! Discover our restaurant's ambiance and latest offers.
    About: Learn more about our story, mission, and values.
    Booking: Reserve your table online for a delightful dining experience.
    Menu: Explore our mouthwatering menu options, from appetizers to desserts.
    Menu Item pages: Dive into detailed descriptions and images of our delicious dishes.
